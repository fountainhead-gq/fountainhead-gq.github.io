Markdown中添加图片的快捷操作流程

  • AutoHotKey
  • Markdown
  • Pyhton
coder

简介说明

在windows环境下编辑markdown时，快速实现复制图片后，调用快捷键上传图片到图库并生成图片链接后添加图片链接的功能。目的只有一个，实现简化添加图片的操作。

流程步骤

  • 注册登录七牛，设置图床
  • 将符合系统版本的AutoHotKey.dll文件保存在Windows/System32文件夹中。 AutoHotkey.dll是用来实现脚本语言对AutoHotkey的调用
  • 测试写好的代码uploadImg_ahk.py

  • 下载安装AutoHotKey,把执行的脚本添加进ahk文件中（run后的路径为绝对路径）,示例如下：

  • 以管理员的身份启动ahk文件

  • 最后测试能否实现功能。
  • 更多内容详见: github

快捷操作

  • 选中并复制想要上传的图片，ctrl + c或右键复制
  • 按下快捷键ctrl + alt + c,此操作实现图片上传至图库，并生成图片链接到剪切板。
  • 按下ctrl + v，把剪切板的链接（格式: ）复制到markdown文件，最终实现简化。

功能预览

在按下快捷键ctrl + alt + c调用程序会短暂的出现CMD窗口，当CMD窗口自动消失后按ctrl + v贴入图片。
