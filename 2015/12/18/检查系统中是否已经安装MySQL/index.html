修改配置文件

在正式启动Mysql以前需要先修改一下配置文件，主要是sudo vi /etc/mysql/my.cnf下，注释掉binding-address=127.0.0.1的这句代码，其他配置根据需要灵活修改即可。

通过 show databases可以查看当前的数据库。关于mysql数据库就行下一步操作，使用use mysql 命令。show tables显示当前数据库的表单。

MySQL管理

启动MySQL服务： sudo /etc/init.d/mysql start

停止MySQL服务： sudo /etc/init.d/mysql stop

重启MySQL服务： sudo /etc/init.d/mysql restart

修改管理员密码： sudo mysqladmin -u root password

取消本地监听： sudo vi /etc/mysql/my.cnf，注释掉bind-address = 127.0.0.1

卸载mysql

su进入root权限，执行/etc/init.d/mysql start/stop关闭服务。
卸载mysql：
sudo apt-get autoremove --purge mysql-server-5.0

sudo apt-get remove mysql-server

sudo apt-get autoremovemysql-server

sudo apt-get remove mysql-common

退出root权限，把#变成$的切换用户命令：ctrl + d，exit，logout
