Go的环境搭建

windows下安装

  • 访问下载页，选择msi安装包。下载好后运行，不用修改默认安装目录 C:\Go\，若安装到其他位置会导致不能执行自己所编写的 Go 代码。安装完成后默认会在环境变量 Path 中添加 C:\Go\bin\，并添加环境变量 GOROOT，值为 Go 安装根目录 C:\Go\ 。
  • 在运行中输入 cmd 打开命令行工具，在提示符下输入 go，检查是否能看到 Usage 信息。输入 cd %GOROOT%，看是否能进入 Go 安装目录。若都成功，说明安装成功。或者输入go version,能显示版本说明安装成功。

科学绘图Matplotlib的简易教程

介绍

Matplotlib是Python最流行的绘图库之一。Matplotlib使用使用起来非常方便，用户可以高度定制绘图模型，允许用户绘制点线图、条线图/直方图、3D图形，甚至是更复杂的图表，比如烛台图表。这个系列由Matplotlib最基本的东西入手，逐渐深入到更具挑战、更漂亮的图表。一旦你熟悉了Matplotlib，你可以通过各种数据类型、数据轴等高度定制自己需要的图表。
附：教程地址

pylab

  1. pylab是matplotlib的模块么，和matplotlib是什么关系?
    pylab 是类似于 matlab 这样一个综合性的平台， pylab 把 Python, NumPy, SciPy, Matplotlib都集合起来了，用的是 IPython 作为界面，把 namespace 都给合到一块。所以pylab 明显不是 matplotlib 的模块。
  2. pylab 是 matplotlib 面向对象绘图库的一个接口。它的语法和 Matlab 十分相近。也就是说，它主要的绘图命令和 Matlab 对应的命令有相似的参数。参见官网
